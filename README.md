# Cosmic Runner 🚀

Một game platformer 2D lấy cảm hứng từ không gian vũ trụ, được viết bằng HTML5 Canvas và JavaScript thuần.

## 🎮 Cách Chơi

### Điều khiển
- **A / ←** : Di chuyển trái
- **D / →** : Di chuyển phải
- **W / SPACE** : Nhảy (có thể nhảy 2 lần liên tiếp!)

### Mục tiêu
- Thu thập ngôi sao ⭐ để ghi điểm
- Thu thập hành tinh 🪐 để được nhiều điểm hơn
- Tránh gai đỏ 🔺 và kẻ thù 👾
- Hoàn thành 5 màn chơi để chiến thắng!

### Hệ thống điểm
- Ngôi sao: +100 điểm
- Hành tinh: +500 điểm
- Tiêu diệt kẻ thù: +300 điểm
- Hoàn thành màn: +1000 điểm
- Mỗi 5000 điểm: +1 mạng sống

### Các loại nền tảng
- 🟣 **Tĩnh** : Nền tảng bình thường
- 🔵 **Di động** : Di chuyển theo chiều ngang hoặc dọc
- 🟢 **Bouncy** : Giúp bạn nhảy cao hơn
- 🟠 **Floating** : Di chuyển lên xuống nhẹ nhàng

## ✨ Tính năng

- 🎨 Đồ họa neon đẹp mắt với hiệu ứng glow
- 🌟 Hệ thống particle effects
- 🎵 Âm thanh game sống động
- 👾 Nhiều loại kẻ thù khác nhau
- 📊 Lưu kỷ lục điểm cao
- 🎯 5 màn chơi với độ khó tăng dần
- 📱 Responsive design

## 🛠️ Công nghệ

- HTML5 Canvas
- Vanilla JavaScript
- Web Audio API cho âm thanh
- LocalStorage cho việc lưu kỷ lục

## 🚀 Chạy game

Mở file `index.html` trong trình duyệt bất kỳ để chơi!

```bash
# Sử dụng Python
python -m http.server 8000

# Hoặc mở trực tiếp
# Windows: start index.html
# macOS: open index.html
# Linux: xdg-open index.html
```
